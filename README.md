

This Python project analyzes player performance using NumPy and Pandas, and visualizes runs scored using Matplotlib.

## Dataset
- Player name
- Team
- Matches played
- Total runs

## Features
- Calculates batting average
- Displays team-wise total runs
- Bar chart visualization of player runs

## Libraries Used
- numpy
- pandas
- matplotlib

## How to Run
1. Install required libraries:
   pip install numpy pandas matplotlib
2. Run the file:
   python code.py# sumit
